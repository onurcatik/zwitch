# Zwitch

## Overview

In this project, I will build a web-based streaming platform similar to Twitch using the MERN stack. I will gain practical experience in setting up user authentication, integrating with OBS for live broadcasting, and implementing real-time chat functionality.

## What I Will Learn

1. **Understand the MERN Stack**: Gain foundational knowledge of MongoDB, Express, React, and Node.js.
2. **Build a Streaming Platform**: Create a basic version of a web-based streaming platform, similar to Twitch.
3. **Implement User Authentication**: Set up user login and registration systems.
4. **Integrate with OBS**: Learn to connect a platform with OBS for live broadcasting.
5. **Add Real-time Chat**: Create a chat system using Socket.IO for user interactions.
6. **Test APIs with Postman**: Develop and test platform APIs to ensure functionality.
7. **Develop with React**: Understand how to use React for building interactive web interfaces.

## Getting Started

### Prerequisites

- Node.js installed on my machine
- MongoDB installed and running
- Basic understanding of JavaScript and React

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/onurcatik/zwitch.git
    cd zwitch
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Set up environment variables:

## Project Structure

- `server/` - Contains the Express server and API routes.
- `client/` - Contains the React front-end application.
- `models/` - Contains the MongoDB models.
- `controllers/` - Contains the logic for handling requests.
- `routes/` - Contains the API routes.
- `public/` - Contains static files.

## Features

- User registration and login
- Live video streaming with OBS integration
- Real-time chat with Socket.IO
- API testing with Postman
